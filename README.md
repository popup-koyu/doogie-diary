# 📓 Doogie Diary

**레트로 게임 스타일의 인터랙티브 다이어리**

1985년 DOS 시대 감성을 재현한 웹 기반 다이어리 애플리케이션입니다.

---

## ✨ 주요 기능

### 🎮 레트로 게임 스타일 UI
- DOS 시대 느낌의 그린 모니터 디자인
- Press Start 2P 폰트 사용
- 픽셀 아트 스타일 그래픽
- 8비트 게임 인터페이스

### 💾 디스켓 크레딧 화면
- 회전하는 별 애니메이션
- NYC 스카이라인 배경
- 캐릭터 소개 그리드
- 1985 → 2025 타임 패러독스 이스터에그

### 📝 다이어리 기능
- 일기 작성 및 저장
- 로컬 스토리지 활용
- 검색 및 필터링
- 타임라인 뷰

---

## 🚀 시작하기

### 실행 방법
```bash
# 1. 파일을 브라우저에서 열기
open doogie-diary.html

# 또는 로컬 서버 실행
python -m http.server 8000
# http://localhost:8000/doogie-diary.html
```
