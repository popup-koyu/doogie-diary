### 2.3 DOOGIE만의 강점 (경쟁사 없음)

```
1. DOS/CRT 레트로 미학
   → 시장에 유사 서비스 없음. 완전한 블루오션.

2. 이미지 픽셀아트 변환
   → 사진을 4단계 녹색 팔레트로 변환하는 기능은 독점적.

3. Mermaid 다이어그램 지원
   → 어떤 일기 앱도 다이어그램을 지원하지 않음.

4. 크리스마스 모드 (8-bit BGM + 눈 효과)
   → 시즌 이벤트를 가진 일기 앱은 전무.

5. 인터랙티브 크레딧 (NYC 스카이라인)
   → "게임" 같은 경험을 주는 일기 앱 없음.

6. 완전 무료 (광고 없음, 유료 플랜 없음)
   → 대부분 경쟁사는 $20~$35/년 구독 모델.

7. Single HTML File 아키텍처
   → 기술적으로 독특한 접근 방식.
```

### 2.4 DOOGIE의 약점 (경쟁사 대비)

```
치명적 부재:
  ❌ 검색 기능 없음 → 과거 일기 찾기 불가
  ❌ 내보내기 없음 → 데이터 이동 불가 (사용자 잠금)
  ❌ 오프라인 모드 없음 → 인터넷 필수

중요한 부재:
  ❌ AI 기능 없음 → 2025-2026 시장 트렌드와 괴리
  ❌ 무드 트래킹 없음 → 기본 기대 기능
  ❌ 스트릭/게이미피케이션 없음 → 리텐션 도구 부재
  ❌ 태그/검색 없음 → 정리/관리 기능 부재

바람직한 부재:
  ❌ "이 날의 기억" 없음
  ❌ 날씨/위치 자동 태깅 없음
  ❌ E2E 암호화 없음
```

### 4.1 DOOGIE의 경쟁 우위

```
DOOGIE의 해자(Moat): DOS 미학 자체가 해자

경쟁사가 DOS 테마를 쉽게 따라할 수 없는 이유:
  1. CRT 노이즈, 스캔라인, 깜빡임 → 기술적 완성도
  2. 녹색 4단계 픽셀아트 변환 → 독자적 알고리즘
  3. 8-bit BGM 생성 → Web Audio API 활용
  4. 인터랙티브 크레딧 → "게임 같은" 경험
  5. 시즌 이벤트 (크리스마스) → 정서적 연결

"레트로 감성의 일기 앱"이라는 카테고리에서
DOOGIE는 유일한 플레이어
```
